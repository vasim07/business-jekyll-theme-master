A brief about ETL/ELT

With the ever increasing size of data, a common problem organization face is to collect data from multiple sources and formats. These raw data often requires collating, cleaning, transformation and standardization. Several tools and technologies (some listed benath) can be used by organization to resolve this problem. No matter which tool is used, but at the core they involve

  • Extract raw data, usually through multiple sources
  • Transform them using domain knowledge
  • Load them into an information hub

    • These stages combined are commonly known as ETL or data pipeline. Some experienced visitor may also notice the business context - converting raw data to information. Traditionally, ETL projects were sufficient, however data revolution brought in some challenges. To overcome, the industry has shifted gears to adopt a modern approach known as ELT.
